# Index Heap
索引堆
> Heap中的上浮和下沉涉及到元素的交换<br/>
> 元素对象可能会很大，交换起来麻烦，索引堆解决这种场景<br/>
> 索引堆不移动元素，对元素的索引进行堆化，堆化后的结果类似二级索引

| 索引 | data | indexes |
| :- | :- | :- |
| 0 | 30 | 6 |
| 1 | 10 | 3 |
| 2 | 20 | 4 |
| 3 | 40 | 1 |
| 4 | 60 | 0 |
| 5 | 50 | 2 |
| 6 | 70 | 5 |
